== Description ==

A very easy to use plugin which lets you chnage the default Wordpress email from name and address.
Has a great admin panel which lets you configure and set your options and also turn the plugin on or off without shutting
the whole plugin down. 

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `eh-from-mail` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the plugins options page and activate the features you want to use
